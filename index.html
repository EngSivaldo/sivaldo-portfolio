Network Routing & Apache Sandbox

Infraestrutura Realizada

Este portfólio não está rodando apenas na nuvem pública. Ele nasceu dentro de um Laboratório de Redes Privadas (Homelab) simulado localmente.

Configurei um servidor Debian como Gateway/Roteador físico ligando a rede externa (Bridge) a uma rede interna protegida. Nela, o servidor Ubuntu hospeda esta aplicação via Apache Multi-Site (Virtual Hosts), sem exposição direta à internet.

Destaques da Configuração:
  • Isolamento de Rede Interna (Subrede 10.200.0.0/24)
  • Debian Router com NAT e IP Forwarding ativo
  • Deploy automatizado usando SSH Agent Forwarding
